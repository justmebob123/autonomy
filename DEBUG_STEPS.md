### Debug Process
1. Identify the error type (syntax, runtime, logic)
2. Locate the source (file, line, function)
3. Understand the cause
4. Implement fix
5. Verify fix resolves the error
6. Check for regressions

### Common Fixes
- ImportError: Check module installation and paths
- SyntaxError: Check for typos, missing colons/brackets
- TypeError: Check argument types and counts
- AttributeError: Check object types and method names
